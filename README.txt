📄 PDF Table Extractor Web App

🔧 Setup Instructions:

1. Install dependencies:
   pip install -r requirements.txt

2. Run the backend:
   python app.py

3. Open index.html in your browser and change the API URL to your backend server.

🌐 For deployment:
- Use services like Render, Railway, or your own VPS.
- Make sure /upload endpoint is publicly accessible.
